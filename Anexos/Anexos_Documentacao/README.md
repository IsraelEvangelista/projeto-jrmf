Esta pasta contém documentos de referência e exemplares que complementam a documentação técnica dos portais de Vazões Jaguaribe RMF e Portal Hidrológico.

## Estrutura das Pastas

Armazena os relatórios exemplares solicitados (6 tipos previstos, atualmente 11 arquivos recebidos):

- Resenha
- Situação dos Açudes Federais
- Apostes (dia/mês/ano)

- Mapa RMF

Observações:
- Foram recebidos exemplares adicionais: Evolução RMF, Monitoramentos Cocó/Maranguapinho, Resenha diária, Situação dos Açudes Federais.
- Sugerida nomenclatura: <Tipo>.<YYYY-MM-DD ou DD.MM.YYYY>.pdf
- Manter variações de versão com sufixo (ex.: v2) quando aplicável.

- Dashboards dos portais
- Fluxos de navegação
